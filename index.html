these are checkboxes

these are radio-buttons

These are basic HTML elements that have the ability to manage state on your website

When the you click on a checkbox, it gains the state of checked. When it you click it again, it loses that state

Radio buttons are a little different, where if they share the same name, only one radio button can have the state of checked at a time

You can then use the pseudo-selector :checked to modify elements when your checkbox is checked!
